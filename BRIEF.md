## 디자인 요구사항
- **미니멀 + 모던**: 흑백 기반, 포인트 컬러 1개 (민트 #10b981 또는 블루 계열)
- **프레젠테이션 스타일**: 각 섹션이 full-viewport 슬라이드로 보임
- **스크롤 스냅**: CSS scroll-snap으로 슬라이드 전환 효과
- **사진 플레이스홀더**: 이미지 자리에 회색 박스 + "사진 추가 예정" 텍스트
- **반응형**: 모바일에서도 잘 보여야 함
- **애니메이션**: 스크롤 시 fade-in 효과 (IntersectionObserver)
- **폰트**: Pretendard 또는 system-ui
- **키보드 네비게이션**: 위/아래 화살표로 슬라이드 이동
